Welcome to my Fibonacci app

To use the app, just add the number of Fibonacci numbers you want to calculate to the end of the above url.
i.e.: <host>/v1/fib/<num>

where "<host>" is the dns or ip address of host running the app and "<num>" is the number of Fibonacci numbers you want to see.

As a note there is a hard limit of 1000 to avoid memory exhaustion issues.
